# autonomous_driving_CSCI_557

### Installation
1. Setup CARLA environment

```
> make env
```
	
2. Start Carla Environment

In Display mode
```
> make run_display
``` 

In Non Display mode
```
> make run_non_display

```

3. Test Carla Environment
```
> make test
```
